# Go Command Line Interface (CLI)

## Basic commands

```bash
# creates go.mod file
go mod init my_domain/my_app

# executes a code
go run .

# compiles a code (creates an executable binary)
go build
```

## Additional commands

* [gofmt](https://pkg.go.dev/cmd/gofmt)
* [golint](https://pkg.go.dev/golang.org/x/lint/golint)
